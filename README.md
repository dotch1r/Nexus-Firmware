# Nexus-Firmware
🚀 NEXUS Firmware


NEXUS is a custom firmware built for the M5StickC Plus2 powered by the ESP32 platform. The goal of this project is to create a portable, modular and easy-to-use toolkit that combines multiple wireless and system utilities inside a clean and responsive interface. Designed with experimentation and learning in mind, NEXUS provides a compact environment where different features such as WiFi tools, BLE utilities, IR functions and system tools can be accessed from a simple menu driven UI.

The firmware focuses on usability and performance while keeping the interface minimal and fast. The menu system is designed to be easy to navigate using the hardware buttons of the M5StickC Plus2, making the device feel like a small pocket toolkit. Each module is organized in categories so users can quickly access different tools without dealing with complex commands or configurations.

✨ Features

📡 WiFi Utilities  
NEXUS includes WiFi related tools such as network scanning and signal information that allow users to explore nearby wireless networks and understand their environment. These tools are useful for learning about wireless communication and experimenting with ESP32 capabilities.

📶 BLE Utilities  
Bluetooth Low Energy tools allow scanning and viewing nearby BLE devices. This can be helpful for testing, development and exploring BLE devices around you.

📺 IR Tools  
The firmware also supports infrared features using the built-in IR LED of the M5StickC Plus2. This allows the device to interact with IR compatible electronics such as TVs and other appliances.

📁 File System Tools  
Support for LittleFS and SD card features makes it possible to store and manage files directly on the device. This is useful for configuration files, captured data and custom resources.

⏰ System Tools  
Additional system utilities include a real-time clock display, brightness control, sound settings and other configurable options that make the firmware more user friendly.

🎨 User Interface  
NEXUS uses a custom graphical interface designed specifically for the M5StickC Plus2 screen. The UI includes a dynamic menu system, status bar with battery level and time display, and smooth navigation between modules.

⚡ Lightweight and Fast  
The firmware is designed to run efficiently on the ESP32 hardware while keeping the code modular so new tools and features can easily be added in the future.

🎯 Project Goal

The main purpose of NEXUS is to create a portable ESP32 toolkit that allows developers, hobbyists and hardware enthusiasts to experiment with different technologies in a single device. By combining wireless utilities, IR functions and system tools into one firmware, the M5StickC Plus2 becomes a compact platform for learning and development.

🛠 Hardware

• M5StickC Plus2  
• ESP32  
• Built-in display and buttons  
• Built-in IR LED  
• Optional external modules

📌 Status

NEXUS is an actively evolving project. New features, tools and improvements will continue to be added as development progresses.
